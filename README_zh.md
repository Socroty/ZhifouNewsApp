该项目是知否新闻系统的安卓客户端程序。  
你也可以修改代码以实现更多功能，并将其应用于自己的应用程序。
## 使用
### 代码
1. 下载此项目的代码到你的计算机；
2. 将代码导入到Android集成开发环境或其它你喜欢的工具；
3. 修改`src/main/java/com/socroty/zhifounews/SocketClient.java`文件为你自己的服务器配置；
4. 也可以修改其它任何地方，以应用于你自己的应用程序；
5. 运行。
